
Q1: Why might we consider using Git?

Answer: We use Git to track changes in our code over time.
It allowes us to save different versions of our work, collaborate with others
and go back to previous version if we make a mistake.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to healthcare. For example, 
doctors can use data science to analyze patient records and 
predict which patients are at risk of getting a disease.
